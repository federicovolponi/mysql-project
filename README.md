# mysql-project

This project was built during the algorithms course in my bachelor in Electronics engineering.
## About
The project is developed in C++, following the object-oriented programming paradigm consists of a mySQL-like relational database management system. 
The following commands are implemented:
- **CREATE TABLE**
- **DROP TABLE**
- **INSERT INTO**
- **DELETE FROM**
- **UPDATE**
- **SELECT**
- **ORDER BY**

The syntax is the same of the mySQL language.
